# URL Shortener REST API 🎉

REST API and UI for get shorter links.

## 💻 Techs
- Node.js (Express)
- MongoDB
- Jest (Unit Testing)
- Eslint (Linting)

## 📦 Installation

### 📰 Clone the project
```bash
git clone https://github.com/emreunsal1/url-shortener.git
```
### 🔻 Install Dependencies
With npm:
```bash
cd ./url-shortener && npm install
```
### 🏃‍♂️ Start the project
With npm:
```
npm run start
```
